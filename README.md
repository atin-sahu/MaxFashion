# Max-Fashion

Max-Fashion is web application that caters clothes to all categories.


## Tech Stack

- HTML
- CSS
- BootStrap
- JavaScript
- LocalStorage


## Features

- Sign-in/Sign-up Page
- Navigation Bar
- Pages For all Categories
- Payment Page
- Cart Page
